La página es muy intuitiva y fácil de manejar.

El usuario puede navegar por distintas secciones en las cuales encontrará información, datos y una calculadora la cual le servirá para ver que tan rentable le resultaría hacer una instalación de paneles solares.

La página ahora cuenta con su propio dominio gracias a heroku, el link de esta es el siguiente: https://tei-sec4.herokuapp.com 

Para usar la página desde github, solo se debe iniciar la terminal de python "wsgi.py", actualmente la página cuenta con una base de datos la cual recopila la información pedida en la calculadora.
Finalmente, no logramos que la página pudiera enviar correos a las personas que los introdujeran en la calculadora pero, ahora tenemos una forma de cuantificar nuestro impacto gracias a la base de datos :)
